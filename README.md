# RecallOS (KnowFlow AI) 🚀

A smart productivity and learning dashboard made to help with memory retention, study flow, and reducing multitasking while learning. Built to feel fast, clean, and easy to use.

## 🔗 Live Deployment

You can check the live version here on Cloudflare:
👉 **[Live Demo Link](https://knowflow-ai-01.matthewstephen291.workers.dev)**

---

## ✨ Features

- **RecallOS Core Engine:** Simple single-file architecture that keeps the UI fast and responsive.
- **Interactive Flashcard System:** Active recall quiz system with score tracking, correct/wrong stats, and a clean result screen.
- **AI Tutor UI:** Helps with course building workflows and breaking down concepts in a easier way.
- **Responsive Navigation:** Layouts are optimized for readability and smooth performance across devices.

## 🛠️ Tech Stack

- **Frontend:** React 19
- **Build Tool:** Vite
- **Routing & State Management:** TanStack Router / TanStack Start
- **Styling:** Tailwind CSS v4 + Lucide Icons
- **Runtime & Package Manager:** Bun
